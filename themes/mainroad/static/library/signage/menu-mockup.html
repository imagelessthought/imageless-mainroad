Help

Groups

Group: Each user must be assigned a group or they can only manage items that they created.

Roles Explained:

Admin Global: Permssion to all items.

Digital-Signage: the DEVICES menu option is only visible and available to those with this role assigned. Group rules still apply.

Asset Manager: assigned for each group separately, not globally. User many be in multile groups but not asset manager for all. Grants permissions to items that require this role.

Image Manager: Grants permission to edit, or delete any image in the libary.

Member: Allows user access to groups playlists and other assets - itmes that do not require asset manager role.

Permissions Explained:

Devices

Register a Device: Digital-Signage; Group Asset Manager

Unregister a Device: Digital-Signage; Group Asset Manager

Assign Playlist: Digital-Signage; Member

Assign Schedule: Digital-Signage; Member

View Current Playlist: Digital-Signage; Member

Lock Channel: Digital-Signage; Group Asset Manager

Channels

Create Channel: Group Asset Manager

Delete Channel: Group Asset Manager

Assign Playlist: Member

Assign Schedule: Member

View Current Playlist: Member

Lock Channel: Group Asset Manager

Kiosks and Presentations may be assigned to a group when created or to the creating user (personal).

Kiosks

Create Kiosk: Group Asset Manager or User

Delete Kiosk: Group Asset Manager or User

Assign Playlist: Member or User

Assign Schedule: Member or User

View Current Playlist: Member or User

Lock Channel: Group Asset Manager (*personal Kiosks cannot be locked - no need)

Presentations

Create Presentation: Group Asset Manager or User

Delete Presentation: Group Asset Manager or User

Assign Playlist: Member or User

Assign Schedule: Member or User

View Current Playlist: Member or User

Lock Channel: Group Asset Manager (*personal Presentations cannot be locked - no need)

Playlists may be assigned to a group when created or to the creating user (personal). Personal playlists may only be assigned to Kiosks and Presentations owned by the user (personal) and not to Devices.

Playlists

Create Playlist: Member or User

Delete Playlist: Group Asset Manager or User

Assign Playlist: Member or User

Assign Schedule: Member or User

View Current Playlist: Member or User

Lock Playlist: Group Asset Manager (*personal Playlists cannot be locked - no need)

The role Image Manager exists to cleanup the library. It will have the ability to manage any image.

Images

Upload Image: All Users : create option to assign to group or personal

Remove and Image: Group Asset Manager or User

Library Management: View all images, assign and remove tags (can only remove tags you assigned unless you are the asset manager of the group that is assigned to it.) The URL to the image should be shown as well.

URLS

Should produce a list of the URLS for all devices, channels, kiosk, presentations, and playlists. For all but playlists the playlist that is assigned should be shown.

Online Services

Postman: API Platform

Postman is an API platform for building and using APIs. Postman simplifies each step of the API lifecycle and streamlines collaboration so you can create better APIs—faster.

GitHub: Version Control

GitHub is a platform and cloud-based service for software development and version control, allowing developers to store and manage their code. It uses Git software, providing the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.

Discord

Discord is a social-media collaboration platform with servers organized into channels by topic. Collaborate and communicate with instant messaging, VOIP, and video chats.

Microsoft 365: Online Access to Applications and Collaboration Tools

Microsoft 365 is a product family of productivity software, collaboration and cloud-based services developed by by Microsoft. It encompasses online services and applications formerly part of the Microsoft Office Suite: Word, Excel, PowerPoint, Outlook, among others. Use the link above to login to your Microsoft 365 account, this will launch the online version of Microsoft Outlook: The other available applications can be accessed by using the icon at the top left.

Microsoft: Azure Cloud Computing

Microsoft Azure is a cloud computing platform which offers access, management, and development of applications and services through global data centers.
